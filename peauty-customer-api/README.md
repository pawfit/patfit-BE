# Customer API Module

Customer API Module은 Peauty 시스템에서 고객용 API를 제공하는 모듈입니다.
Customer API Module의 주요 책임은 오직 고객용 API 기능 개발과 제공에 집중하는 것입니다.

## 주요 기능

- **회원 관리**: 유저 회원가입, 로그인, 개인정보 관리 등 회원 관련 API를 제공합니다. 회원 인증, 프로필 조회 및 수정, 비밀번호 변경 등의 기능을 지원합니다.

- **반려견 정보 관리**: 반려견 등록, 정보 조회, 수정, 삭제 등의 API를 제공합니다. 

- **디자이너 탐색 및 예약**: 디자이너 정보 조회, 검색, 필터링 등의 API를 제공합니다.선택한 디자이너와의 예약을 생성하고 관리하는 기능을 제공합니다.

- **반려견 미용 프로세스 관리**: 반려견 미용 입찰 프로세스와 관련된 API를 제공합니다. 미용 서비스 예약, 진행 상황 추적, 완료 확인 등의 기능을 지원합니다.
  
- **결제 처리**: 고객의 결제 관련 API를 제공합니다. 
  
- **리뷰 및 평점 관리**: 고객이 디자이너나 미용 서비스에 대한 리뷰를 작성하고 평점을 남길 수 있는 API를 제공합니다.
